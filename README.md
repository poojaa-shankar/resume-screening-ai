# Resume Screening AI (Python, NLP)

This project automatically ranks resumes against a job description using NLP techniques.

## Features
- Text preprocessing
- TF-IDF vectorization
- Cosine similarity for ranking
- Supports multiple resumes

## Technologies Used
- Python
- scikit-learn
- NLP (TF-IDF, cosine similarity)

## How It Works
1. Resume and job description text is cleaned
2. Converted to numerical vectors using TF-IDF
3. Cosine similarity is calculated
4. Resumes are ranked by relevance

## Use Case
Helps recruiters shortlist candidates efficiently.
